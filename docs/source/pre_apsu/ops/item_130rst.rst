Video recording
===============

.. contents:: 
   :local:

To obtain comoputer screen video recording use::

   tomo@handyn ~ $ conda install -c orodbhen simple-screen-recorder
   tomo@handyn ~ $ simplescreenrecorder


You can video record an experiment with::

   tomo@handyn ~ $ simplescreenrecorder
